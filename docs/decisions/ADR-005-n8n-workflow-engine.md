# ADR-005: n8n Workflow Engine

**Date**: 2026-02-01  |  **Status**: Accepted

### Decision
Use n8n as the workflow automation engine for scheduled tasks, agent pipelines, and notification routing.

### Consequences
- ✅ Visual workflow builder
- ✅ Self-hosted (data privacy)
- ✅ 400+ integrations (WhatsApp, email, Slack)
- ✅ Cron job support for scrapers
