# ADR-004: Opus for Planning, Gemini for Coding

**Date**: 2026-02-01  |  **Status**: Accepted

### Decision
Use Claude Opus 4.6 for strategic planning and architecture. Use Gemini 3.1 Pro for code generation and implementation.

### Consequences
- ✅ Best-in-class planning quality (Opus)
- ✅ Fast, cost-effective coding (Gemini)
- ✅ Each model plays to its strengths
