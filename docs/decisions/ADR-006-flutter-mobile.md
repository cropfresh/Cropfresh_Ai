# ADR-006: Flutter for Mobile

**Date**: 2026-02-01  |  **Status**: Accepted

### Decision
Use Flutter for the CropFresh mobile app (farmer + buyer).

### Consequences
- ✅ Single codebase for Android + iOS
- ✅ Fast development with Dart
- ✅ Native performance
- ✅ Firebase integration
- ⚠️ Team needs to learn Dart
