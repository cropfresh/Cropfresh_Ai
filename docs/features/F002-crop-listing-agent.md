# F002: Crop Listing Agent

## Overview
AI agent that helps farmers create crop listings from photos and voice.

## User Story
As a farmer, I want to send a photo of my crop and have AI create a listing automatically.

## Acceptance Criteria
- [ ] Classify crop type from photo (>90% accuracy)
- [ ] Assess quality grade (A/B/C)
- [ ] Suggest fair price based on market data
- [ ] Support Kannada voice input
