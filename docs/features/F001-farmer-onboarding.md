# F001: Farmer Onboarding

## Overview
Simple phone-based registration with OTP verification.

## User Story
As a farmer, I want to register with just my phone number so I can start listing crops quickly.

## Acceptance Criteria
- [ ] OTP sent via SMS within 5 seconds
- [ ] Profile created with minimal info (name, phone, district)
- [ ] Kannada language selected by default in Karnataka
- [ ] Tutorial flow on first login
