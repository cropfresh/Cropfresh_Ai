# User Journeys

## Journey 1: Farmer Lists Crop
1. Farmer opens WhatsApp / CropFresh app
2. Sends voice message: "ನನ್ನ ಟೊಮ್ಯಾಟೊ 50 ಕೆಜಿ ಮಾರಬೇಕು" (I want to sell 50kg tomatoes)
3. Bot responds in Kannada: "ಫೋಟೋ ಕಳಿಸಿ" (Please send photo)
4. Farmer sends crop photo
5. AI Agent: classifies crop, assesses quality (Grade A), predicts price (₹35/kg)
6. Bot: "ನಿಮ್ಮ ಟೊಮ್ಯಾಟೊ Grade A, ₹35/ಕೆಜಿ. ಲಿಸ್ಟ್ ಮಾಡಲಾ?" (Your tomato is Grade A, ₹35/kg. List it?)
7. Farmer confirms → Listing created

## Journey 2: Buyer Finds & Orders
1. Buyer opens CropFresh app
2. Searches: "Grade A tomatoes near Bangalore, 200kg"
3. AI Matching Agent finds 3 farmer matches
4. Buyer reviews listings with photos, grades, prices
5. Selects farmer Raju's listing
6. Places order → UPI payment (escrow)
7. Logistics arranged → Delivery tracked
8. Buyer confirms receipt → Payment released to farmer

## Journey 3: Price Discovery
1. Farmer asks (voice): "ಇಂದು ಟೊಮ್ಯಾಟೊ ಬೆಲೆ ಎಷ್ಟು?" (What's tomato price today?)
2. Price Agent fetches APMC data from Hubli, Belgaum, Bangalore mandis
3. Analyzes: weather impact, seasonal trend, supply-demand
4. Responds: "ಇಂದು ₹32-38/ಕೆಜಿ, ಮುಂದಿನ ವಾರ ₹40+ ಆಗಬಹುದು" (Today ₹32-38/kg, may reach ₹40+ next week)
