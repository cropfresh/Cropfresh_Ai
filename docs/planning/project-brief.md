# Project Brief — CropFresh AI

## What is CropFresh?
CropFresh is an AI-powered agricultural marketplace that connects farmers in Karnataka, India directly with buyers. It eliminates middlemen, provides real-time market intelligence, and operates through voice-first Kannada interaction.

## Why Does CropFresh Exist?
Indian farmers lose 30-40% of crop value to intermediaries. CropFresh uses AI agents to automate crop listing, quality assessment, price prediction, and buyer matching — giving farmers better prices and buyers reliable supply.

## How It Works
1. **Farmer lists crop** → AI agent processes photo + voice → Creates listing
2. **Price prediction** → Real-time APMC data + weather + history → Fair price
3. **Quality grading** → Computer vision → A/B/C grade
4. **Buyer matching** → Graph-based matching → Best buyer found
5. **Transaction** → UPI payment → Logistics tracking

## Key Differentiators
- Voice-first Kannada interaction (WhatsApp + App)
- AI-powered quality assessment (no subjective grading)
- Real-time price discovery (APMC + eNAM data)
- Direct farmer-buyer connection (no middlemen)
